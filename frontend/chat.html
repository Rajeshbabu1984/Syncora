      How to call a webhook

      POST to the webhook URL from any script, CI/CD, or automation:

      curl -X POST https://your-server/bots/webhook/YOUR_TOKEN \
  -H "Content-Type: application/json" \
  -d '{
    "content":    "Build passed ✅",
    "channel_id": 1,
    "open_url":   "https://youtube.com",
    "shell_cmd":  "start notepad.exe",
    "url_target": "self"
  }'

      url_target: "self" = URL opens only in the bot owner's browser  | "channel" = opens for everyone (bot owner must be the channel owner)
      open_url — opens a URL in browser  | shell_cmd — server-side command (e.g. start calc.exe)
